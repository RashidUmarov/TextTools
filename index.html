Описание

Composer — «Переформатирование списка»

На странице находится два текстовых поля:

  • Левое — ввод исходного HTML со списками;

  • Правое — вывод результата.
    Кнопка Look for list проверяет наличие списков, затем кнопка Compose links запускает функцию composeHTML(), которая разбивает или объединяет списки по заголовкам (реорганизация <ul>/<li>).


Sizer — «Масштабирование картинок»

Позволяет изменить размер всех тегов <img> в HTML-коде на заданный процент.

  • Поле ввода New size (%) задаёт коэффициент масштабирования.

  • При нажатии Scale and copy → весь блок HTML обрабатывается JavaScript-функцией (встроенный скрипт), которая пересчитывает атрибуты width/height и выводит обновлённый код в правое поле.
    Кроме того, под блоками кода отображается панель с «До»/«После» для каждой найденной картинки.


Changer — «Замена картинок»

Инструмент для массовой замены атрибутов у <img>:

  1. Look for img — сканирует левый и правый текст и проверяет наличие картинок.

  2. Replace SRC — меняет только src у изображений (функция replaceSrcInImg() в changer.js).

  3. Replace SRC + Width + Hight — одновременно обновляет src, width и height (функция replaceImg()).


Blocks — «Замена блоков кода»

Ищет блоки <pre class="code">…</pre> в HTML и заменяет их содержимое по заданному алгоритму.

  • Look for tag pre — проверка наличия таких блоков;

  • Replace Code Blocks — выполняет замену через replacePreBlock() в code_blocks.js.


Cleaner — «Чистка HTML»

Удаляет пустые теги <div> и <span>, не содержащие атрибуты class или style.

  • Check HTML — определяет, есть ли такие теги;

  • span delete и div delete — соответственно удаляют пустые <span> и <div> через функции deleteEmptySpans() и deleteEmptyDivs() в cleaner.js.
